# 韩中漫画批量翻译 操作指南

## 概述

韩中漫画批量翻译工具用于将韩文漫画（Webtoon/Manhwa）批量翻译为简体中文。

- **翻译器**：Qwen3 14B (Ollama)
- **翻译方向**：韩文 → 简体中文
- **无降级链**：Ollama 不可达时直接报错退出

## 环境准备

### 1. Ollama 服务

确保 Ollama 服务在 `192.168.1.15:11434` 运行，且已拉取模型：

```bash
# 拉取 Qwen3 14B 模型
ollama pull qwen3:14b-q4_k_m

# 验证模型可用
curl http://192.168.1.15:11434/api/tags
```

### 2. Conda 环境

使用 `TraeAI-2` 环境（与日中翻译共用）：

```bash
conda activate TraeAI-2
```

### 3. 环境变量（已预设）

`.command` 脚本已预设以下环境变量：

```bash
export CUSTOM_OPENAI_API_BASE='http://192.168.1.15:11434/v1'
export CUSTOM_OPENAI_MODEL='qwen3:14b-q4_k_m'
export CUSTOM_OPENAI_API_KEY='ollama'
```

## 使用方式一：Finder 双击启动（推荐）

1. 在 Finder 中找到 `批量韩中翻译.command`
2. 双击启动 → 终端窗口自动打开
3. 输入漫画目录路径（如 `/Users/xxx/Downloads/漫画名/韩文`）
4. 确认是否重新翻译（输入 `y` 全量重新翻译，其他键续传）
5. 程序自动执行，完成后输出总结

**输出目录**：输入 `韩文` → 输出 `韩文 汉化`（位于输入目录的同级目录）

## 使用方式二：命令行启动

```bash
conda activate TraeAI-2
python -m manga_translator.batch_ko
```

## 翻译流程

```
输入: 韩文/                    ← 用户输入的漫画目录
      ├── 001.webp            ← 当前目录的图片（优先翻译）
      ├── 002.webp
      ├── 01/                 ← 子目录
      │   ├── page01.webp
      │   └── page02.webp
      └── 02a/                ← 子目录
          └── 001.webp

执行顺序:
  1. 加载翻译模型（一次性，约 10s）
  2. 探测 Ollama 服务可达性
  3. 翻译 韩文/ 的图片
  4. 翻译 韩文/01/ 的全部图片
  5. 翻译 韩文/02a/ 的全部图片

输出: 韩文 汉化/               ← 同级目录，"原名 汉化"
      ├── 001.webp
      ├── 002.webp
      ├── 01/
      │   ├── page01.webp
      │   └── page02.webp
      └── 02a/
          └── 001.webp
```

## 子目录排序规则

与日中翻译相同，详见 CONTEXT.md。

## 中断续传

- 每目录下自动生成 `.translate_progress.json`（进度文件）
- 每翻译成功一张图片就立即写入进度
- 脚本重新执行时自动读取进度，跳过已翻译的图片
- 中断后重启 = 从中断点继续翻译

**重新翻译**：输入目录名后，确认提示时输入 `y` 即清空所有进度重新翻译。

## 支持的图片格式

| 格式 | 扩展名 |
|------|--------|
| WebP | `.webp` |
| PNG | `.png` |
| JPEG | `.jpg`, `.jpeg` |
| BMP | `.bmp` |
| TIFF | `.tiff`, `.tif` |
| GIF | `.gif` |

## 错误处理

| 错误 | 行为 |
|------|------|
| Ollama 服务不可达 | 报错退出，提示检查 Ollama 服务 |
| 目录不存在 | 提示重新输入 |
| 翻译失败 | 报错退出（无降级，attempts=1） |

## 故障排查

### Ollama 不可达

```bash
# 检查 Ollama 服务状态
curl http://192.168.1.15:11434/api/tags

# 如果不可达，在远程机器上启动 Ollama
ollama serve
```

### 模型未找到

```bash
# 拉取模型
ollama pull qwen3:14b-q4_k_m

# 验证
curl http://192.168.1.15:11434/api/tags | grep qwen3
```

### 与日中翻译的区别

| 维度 | 日中翻译 | 韩中翻译 |
|------|---------|---------|
| 启动脚本 | `批量日中翻译.command` | `批量韩中翻译.command` |
| 入口模块 | `manga_translator.batch` | `manga_translator.batch_ko` |
| 翻译模型 | Sakura 14B | Qwen3 14B |
| 降级链 | 方式B(GGUF) → 方式A(Ollama) | 无降级 |
| 滑动窗口 | 已集成 | 未集成 |
| 质量评估 | 已集成 | 未集成 |
| 输出目录 | `原名 汉化` | `原名 汉化`（相同） |
