## 翻译器性能预估

基于 Qwen3 14B Q4_K_M 在 Ollama 上的推理速度，参考日中翻译 Sakura 14B 的实测数据：

| 阶段 | 预估耗时/页 | 占比 |
|------|-----------|------|
| Detection (文字检测) | ~0.3s | 4% |
| OCR (文字识别) | ~1.0s | 14% |
| **翻译 (Translation)** | ~3.0-4.0s | 45-50% |
| Inpainting (文字擦除) | ~1.6s | 23% |
| 渲染+IO | ~1.0s | 14% |
| **总计** | **~7.0-8.0s/页** | **100%** |

> 注：翻译阶段耗时为预估值。Qwen3 14B 与 Sakura 14B 参数规模相近，推理速度应接近。实际耗时可能因 Prompt 长度、文字量、网络延迟等因素波动。

## 优化措施

### 1. 禁用 enableThinking

- **效果**：减少推理延迟约 30-50%
- **实现**：`extra_body = {'enable_thinking': False}`
- **质量影响**：无明显影响（漫画翻译为确定性任务）

### 2. 温度参数优化

- **temperature = 0.3**：降低随机性，提高翻译一致性
- **top_p = 0.3**：限制采样范围，减少翻译偏差

### 3. 模型单次加载

- 模型加载一次（约 10s），整个批次复用
- 避免逐页重复加载开销

### 4. 进度续传

- 每页翻译完成后立即写入进度文件
- 中断后重启自动跳过已翻译页面
